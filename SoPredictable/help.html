Help

This Shiny App is a simple type-ahead predictor. Just type in the input field and it will suggest a probable next word. Click on one of the suggested terms and it will be added to your input. If the suggestion is made in the middle of a word (indicated by a different button color) your word will be completed. You can click on the input again to continue typing or edit your input in any way you want.

Click on the Clear button to reset your input.

The number of suggested terms is capped to the 8 most probable terms of the predicted set.

Word Charts

The Word Charts tab will display a word cloud or a probability table of all terms in the predcition set (probability is normalized for the set), not just the suggested buttons. You can toggle between chart formats. The charts are a good indicator the relative porbability of the terms suggested.

Inner Data

The Inner Data tab will show for each of the levels of the algorithm the number of n-gram matches, a summary of weights, and total weight of each level. When there is no match it will indicate that the algorithm is guessing based on context terms or, when nothing typed can be put in context in the corpus, it is providing a best guess based on word frequency.

Context Boost

When the context toggle is selected, the prediction set is altered based on the context provided by the entire input stream and not just the last three tokens. For example "government proposed to increase..." will look at the last three tokens to suggest a next term, but with the context toggle the entire input is considered, yielding a different prediction. For the effect to be apreciated use terms with heavy significance (government, congress, ...) in long input streams to provide enough context.

Caveat Emptor

  • Prediction will only act on the right-end, not where you type.
  • There is no control over typing beyond the visible end of the input field.
  • Depending on your connection, you can easily outrun the predictor. Slow down if it feels sluggish
  • Remember to set focus on the input field if you want to type again after clicking on a prediction: this is smart, but not so smart.
